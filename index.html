HTML Form Reference

Form Tags

Tag Name Description
<form> Form The wrapper element for all HTML forms.
<input> Input The input element has a type attribute which determines how it renders on the web page and what kind of data it can accept.
<label> Label The label tag defines a label for many form elements.
<textarea> Textarea The textarea element is used to create a bigger text field for users to write more text.
<select> Select Defines a drop-down list.
<option value =""> Option value The <option> elements defines an option that can be selected.
<datalist id=""> Datalist input The <datalist> is used with an <input type="text"> element. The <input> creates a text field that users can type into and filter options from the <datalist>.

Form Attributes

Attribute Name Which element Description
type="text" Text fields <input> <input type="text"> defines a single-line input field for text input.
name="" Name <input> We must include a name attribute for the <input> — without the name attribute, information in the <input> won’t be sent when the <form> is submitted.
for="" For <label> & <input> The for attribute of the <label> tag should be equal to the id attribute of the <input> element to bind them together.
type ="password" Password <input> An <input type ="password"> element will replace input text with another character like an asterisk (*) or a dot (•).
type="number" Number <input> By setting type="number" for an <input>we can restrict what users type into the input field to just numbers.
step="" Step number attribute in <input> We can provide a step attribute which creates arrows inside the input field to increase or decrease by the value of the step attribute.
type="range" Range input <input> Another option we could use is setting type to "range" which creates a slider. - You can add assign min & max values & step.
type="checkbox" Checkbox <input> A checkbox input; use value="name of value" to create the checkbox name
type="radio" Radio button <input> Multiple options - radio buttons (like checkboxes) do not display their value. We have an associated <label> to represent the value of the radio button. To group radio buttons together, we assign them the same name and only one radio button from that group can be selected.
type="submit" Submit form <input> <input type="submit" value="Send"> - if not using value attribute, the default text is submit.
rows="" cols="" rows & columns <textarea> The rows attribute specifies the visible number of lines in a text area. The cols attribute specifies the visible width of a text area.
required Required <input> Used to make an input mandatory to fill out.
